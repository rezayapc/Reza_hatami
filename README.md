# V2Ray VPN Configuration - VLESS Protocol

## 📋 مشخصات کنفیگ
- **پروتکل:** VLESS
- **پورت:** 1080
- **تناسب:** اندروید + لینوکس سرور
- **نوع:** سریع و پایدار

## 🚀 برای استفاده:

### 1️⃣ روی سرور لینوکس:
```bash
bash install.sh
```

### 2️⃣ روی اندروید:
- اپ **v2rayNG** یا **v2rayNGX** را نصب کنید
- فایل `client-config.json` را import کنید
- یا QR code را scan کنید

---

## 📁 فایل‌ها:
- `server-config.json` - تنظیمات سرور
- `client-config.json` - تنظیمات اندروید
- `install.sh` - اسکریپت نصب
